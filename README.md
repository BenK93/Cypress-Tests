# Cypress-Tests
## Installation
  make sure to have nodejs on your computer
  1. Clone repo: 
  2. cd to repo
  3. npm install (dependencies)
  4. npx cypress open (viewing tests)
  
## Tests:
  1. Testing creation of shapes
  2. Testing Deletion of shapes
  3. Testing dragging of shapes
  4. Testing editting of shapes
  5. Testing splitting of Polygon
  6. Testing Zoom In/Out button
